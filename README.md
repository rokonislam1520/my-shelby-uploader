# My Shelby Uploader

Simple file uploader built with **Shelby** – Verifiable Global Object Storage for AI.

**Made by Rokon** during Shelby Early Access.

## Features
- Upload images/videos to Shelby network
- "Write once, read anywhere"
- Cryptographic proof (future real version e)

## How to Run
1. Open `index.html` file in any browser
2. Select a file and click "Upload to Shelby"

- HTML + JavaScript
- Ready for Shelby SDK integration
