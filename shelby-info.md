# About Shelby

Shelby is a verifiable global object storage platform co-developed by **Aptos Labs** and **Jump Crypto**.

**Core Features:**
- Single global namespace → No data replication
- Cryptographic proof for every read
- Up to 70% lower egress cost
- High performance hot storage (sub-second reads)
- S3-compatible API
- Perfect for AI datasets, GenAI content, and creator economy

This project is made to showcase my understanding and interest in building on Shelby.

Learn more: https://shelby.xyz
