# How to Deploy This Project Live

### Easy Deployment Options:

1. **Vercel (Recommended - Free)**
   - vercel.com e login with GitHub
   - Import this repo (`my-shelby-uploader`)
   - Deploy → live link pabe (e.g. my-shelby-uploader.vercel.app)

2. **GitHub Pages**
   - Repo settings e GitHub Pages enable koro
   - Source: main branch / root
   - Live: https://YOUR-USERNAME.github.io/my-shelby-uploader

3. **Netlify** (Drag & drop index.html o possible)

After deploy, README.md e live demo link add koro.

Example live link: (pore add kora hobe)

This makes the project more impressive for profile.
