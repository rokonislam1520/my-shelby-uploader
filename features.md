This project demonstrates the power of **Shelby** — a next-generation verifiable global object storage platform co-developed by **Aptos Labs** and **Jump Crypto**.

### Core Features of Shelby

1. **Single Global Namespace**
   - Data ekbar upload koro (ekta address e)
   - Pore jekono region theke read kora jabe — kono replication lagbe na
   - "Write Once, Read Anywhere" philosophy

2. **Cryptographic Proofs on Every Read**
   - Prottek read er sathe cryptographic receipt asbe
   - Ownership, provenance, consent, and attribution automatically verified
   - AI training e proprietary datasets use korar jonno perfect (audit easy)

3. **Dramatically Lower Costs**
   - Traditional cloud (AWS, GCP) er theke expected **70% kom egress cost**
   - Scale e beshi data use korleo khoroch beshi barbe na
   - Pay-per-read model with incentives for providers

4. **High Performance Hot Storage**
   - Sub-second reads with low latency
   - High throughput
   - Real-time AI inference, video streaming, GenAI content delivery er jonno ideal
   - Not cold storage — always hot and fast

5. **S3-Compatible API**
   - Existing tools and libraries er sathe easy integration
   - No major code change lagbe

6. **Blockchain-Powered Verifiability**
   - Aptos blockchain use kore coordination and finality
   - Cryptoeconomic security
   - Chain-agnostic support (Ethereum, Solana, React SDK available)

7. **Built for AI & Creator Economy**
   - Secure sharing of proprietary AI datasets
   - Verifiable GenAI content distribution
   - Token-gated content
   - AI agents er jonno data layer (Shelby MCP Server + Skills)
